# 02 - Estruturas de Dados e Algoritmos

Exercícios práticos sobre estruturas de dados em Java, organizados por
tipo de estrutura. Partindo de arrays simples, evoluindo para listas,
filas, pilhas e estruturas mais complexas como árvores e grafos.

## Organização

### 01 - Arrays
| Pasta | Conceito |
|-------|----------|
| `01-Arrays_01` | Declaração, atribuição e iteração |
| `01-Arrays_02` | Busca de maior e menor elemento |
| `01-Arrays_03` | Soma e cálculo de média |
| `01-Arrays_04` | Ordenação com Bubble Sort |
| `01-Arrays_05` | Introdução conceito de Vetor encapsulado em classe |


## Conceitos abordados até agora
- Declaração e manipulação de arrays
- Busca e ordenação manual (Bubble Sort)
