# Estudos em Java

Repositório com exercícios práticos da minha jornada de aprendizado em Java,
construído de forma contínua conforme avanço nos estudos. O conteúdo é fruto
de diversas fontes — cursos, vídeos e pesquisas — consolidados em exercícios
próprios para fixação de cada tema.

## Fontes de estudo
- [Maratona Java Virado no Jiraya — DevDojo](https://youtube.com/playlist?list=PL62G310vn6nFIsOCC0H-C2infYgwm8SWW)
