# Estudos em Java

Repositório com exercícios práticos da minha jornada de aprendizado em Java,
construído de forma contínua conforme avanço nos estudos. O conteúdo é fruto
de diversas fontes — cursos, vídeos e pesquisas — consolidados em exercícios
próprios para fixação de cada tema.

- Vídeos, canais e artigos diversos da internet

## Conteúdo

O repositório é organizado em pastas por tema, crescendo continuamente com o avançar dos estudos.

| Pasta | Descrição | 
|-------|-----------|
| `01-Fundamentos` | If/else, loops, switch
| `02-Estrutura de Dados e Algoritmo` | Arrays, Matrizes, Filas, Pilhas, etc... 
| `03-Orientação a Objeto` | Abstraçao, Herança, Encapsulamento e Polimorfismo 


## Tecnologias
- Java 17
- VS Code

## Sobre os Exercícios
Os exercícios foram desenvolvidos por mim com o objetivo de praticar e fixar
os conceitos estudados. Fazendo e refazendo exercícios vindos dos cursos e videos para uma melhor compreensão de seu funcionamento.

## Termos 
Nos arquivos pode-se ver nomes como "Main_S01L01", nomes assim servem para referenciar a qual Sessão (S) o arquivo pertence e qual Lição (L) ele é.
No caso, o exemplo citado refere-se ao arquivo da Sessão 1 - Lição 1.
